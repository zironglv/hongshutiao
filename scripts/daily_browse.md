# 小红书养号任务

## 任务目标

使用 Chrome MCP 服务，模拟正常用户浏览小红书，实现养号目的。

## 执行频率

- 每天 1 次
- 随机时间：上午 9:00-11:00 或 下午 14:00-16:00 或 晚上 19:00-21:00
- 每次浏览 5-15 分钟

## 执行步骤

### 第一步：打开小红书

```
使用 chrome MCP:
1. 打开浏览器（如果未打开）
2. 导航到 https://www.xiaohongshu.com
3. 等待页面加载
```

### 第二步：随机浏览首页

```
1. 在首页随机滚动 2-3 次
2. 停留在任意帖子 3-5 秒
3. 模拟正常用户的随机行为
```

### 第三步：搜索关键词

从以下关键词中随机选择 2-4 个：

**核心关键词**：
- 红利指数
- 红利低波
- 股息率
- 红利ETF

**扩展关键词**：
- 定投红利
- 红利基金
- 高股息
- 稳健投资
- 被动收入
- 分红投资

**长尾关键词**：
- 红利指数定投
- 中证红利
- 上证红利
- 红利投资策略

### 第四步：浏览搜索结果

```
每个关键词：
1. 搜索后随机滚动 2-3 次
2. 点击进入 2-5 个帖子
3. 在每个帖子内：
   - 停留 10-30 秒
   - 随机滚动浏览
   - 以 30% 概率点赞
   - 以 20% 概率收藏
   - 以 10% 概率查看评论
```

### 第五步：记录学习内容

```
浏览过程中，注意记录：
1. 热门标题格式
2. 高赞帖子特点
3. 用户讨论热点
4. 常见问题

保存到: hongshutiao/logs/learned_content.json
```

### 第六步：关闭

```
1. 随机等待 1-3 分钟
2. 关闭浏览器标签或保持最小化
```

## 行为规范

### 要做的 ✅

- 随机时间间隔（模拟人类节奏）
- 鼠标滚动自然
- 停留时间合理
- 偶尔返回首页
- 行为模式随机化

### 不要做的 ❌

- 不要快速连续点击
- 不要固定时间间隔
- 不要浏览太多帖子（单次不超过 20 个）
- 不要频繁搜索相同关键词
- 不要在非工作时间浏览

## 异常处理

- 如果遇到登录页面：停止任务，记录状态
- 如果页面加载失败：等待 30 秒后重试，最多 3 次
- 如果浏览器崩溃：记录错误，下次重新开始
